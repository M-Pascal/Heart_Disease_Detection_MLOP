Model Performance Visualizations

Class Distribution

The Class Distribution graph represents the distribution of the target variable, showing the number of patients with and without heart disease. The dataset appears nearly balanced, which is important for training models effectively without bias towards one class.

Logistic Regression Model

This Confusion Matrix visualizes the actual vs. predicted classifications for heart disease. While the model correctly identifies many cases, it also misclassifies some patients, indicating areas where improvement may be needed. Despite its simplicity, logistic regression provides a solid baseline for comparison.

Random Forest Model

The Random Forest Model Confusion Matrix displays the classification results using the Random Forest algorithm. This model demonstrates a stronger ability to differentiate between heart disease and non-heart disease cases compared to logistic regression, with fewer misclassifications. This suggests that the ensemble learning approach of Random Forest is beneficial for this task.

Neuron Network Model

This Neural Network Model Confusion Matrix provides an evaluation of a deep learning approach in predicting heart disease. The neural network shows improved classification performance over logistic regression, with fewer misclassified cases. This indicates that the model is learning complex patterns in the data, which enhances its predictive capabilities.

Loss Curve (NN)

The Loss Curve for the Neural Network tracks the training and validation loss over multiple epochs. The downward trend in both curves suggests that the model is learning effectively, reducing errors with each epoch. However, the slight gap between the training and validation loss towards the end hints at potential overfitting, meaning the model might be performing slightly better on training data than on unseen validation data.
